# AISEA Week 2025

Southeast Asia's largest grassroots AI builder movement.

**Event Dates:** 24-30 November 2025

## About

AISEA Week brings together 900+ AI builders, startups, corporates, and VCs from across Southeast Asia. This is where the region's AI ecosystem connects, and where partners help turn potential into impact.

## Features

- Dithered wave shader background
- Chromatic dispersion glass effects
- Responsive design
- Built with Next.js 16 and React Three Fiber

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 16
- React Three Fiber
- Framer Motion
- Tailwind CSS 4
- TypeScript
